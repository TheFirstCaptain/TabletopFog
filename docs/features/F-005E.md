# F-005E: Campaign Card Presentation

## Objective

Improve the visual presentation of campaign cards.

## User Value

The Campaign Library feels calmer, easier to scan, and more like browsing a
small tabletop collection than managing records.

## Non-Goals

- Do not change encounter cards.
- Do not change the Encounter Workspace.
- Do not change storage.
- Do not add new functionality.
- Do not change campaign creation or metadata behavior.

## Acceptance Notes

- Campaign card typography is consistent with the design language.
- Card spacing supports quick scanning without crowding.
- Campaign icon placement is visually balanced.
- Description layout is readable and does not dominate the card.
- Metadata layout is quiet and secondary to campaign identity.
- Hover and focus behavior are clear, accessible, and restrained.
- Responsive spacing remains usable on Chromebook-sized and narrow viewports.
- Campaign cards remain cards, not tables or dashboard widgets.
